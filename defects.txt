Файл cmd.html.

2. CD - Изменить каталог
3. Date - Выводит или устанавливает текущую дату
4. Time - Выводит или устанавливает системное время

Файл mac.html.
1. mv - расфшифровывается как "Move", что переводится как "переместить", "передвинуть". С помощью нее можно переместить, переименовать или удалить файл.
2. mkdir -  расшифровывается как "Make directory", что переводится как "создать папку".

Файл git.html
1. git clone — создает локальную копию проекта, который уже существует удаленно. Клон включает в себя все файлы проекта, журнал и ветви.
2. git add — подготавливает изменение.
3. git commit — сохраняет моментальный снимок в журнале проекта и завершает процесс отслеживания изменений.
4. git status — выводит состояние изменений: не отслеживаются, изменены или подготовлены.
5. git push — обновляет удаленный репозиторий с учетом фиксаций, выполненных в ветви локально.
